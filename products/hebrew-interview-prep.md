### 🔷 טראק 1: C# & .NET (10 שיעורים)
**משך כולל:** ~3 שעות | **בעיות:** 50 שאלות

תוכן מלא של קורס הכנה ל-C# שמכסה את הנושאים הקריטיים ביותר לראיון:

1. **יסודות C#** — טיפוסים, ירושה, פולימורפיזם, Interfaces
2. **Collections ו-LINQ** — רשימות, שאילתות, ו-LINQ Performance
3. **Async/Await** — תכנות אסינכרוני, Tasks, ConfigureAwait
4. **Memory Management** — Garbage Collector, IDisposable, Span<T>
5. **Design Patterns** — Singleton, Factory, Observer, Repository, DI
6. **Entity Framework & LINQ to SQL** — DbContext, N+1 Problem, Migrations
7. **ASP.NET Core** — Middleware, DI Container, Controllers, Minimal APIs
8. **Threading & Concurrency** — Thread Pool, Locks, ConcurrentCollections, Channels
9. **Live Coding Challenges** — LRU Cache, LINQ Methods, Thread-Safe Singleton
10. **Mock Interview** — סימולציית ראיון מלאה עם feedback

**בנק של 50 שאלות ראיון** עם תשובות מלאות, דוגמאות קוד, וטיפים לראיון.

### 🔷 טראק 2: אלגוריתמים (10 שיעורים)
**משך כולל:** ~3.5 שעות | **בעיות:** 75 בעיות LeetCode

הכנה מלאה לשאלות אלגוריתמים בראיונות, כולל תרגול חי על בעיות אמיתיות:

1. **Arrays & Strings** — Two Pointers, Sliding Window, Prefix Sum
2. **Hash Maps & Sets** — Frequency Counting, Grouping, Uniqueness
3. **Linked Lists** — Reverse, Merge, Cycle Detection, Floyd's Algorithm
4. **Stacks & Queues** — Monotonic Stack, Valid Parentheses, Priority Queue
5. **Trees & BST** — DFS, BFS, Level Order, Validation, LCA
6. **Graphs** — BFS, DFS, Dijkstra, Topological Sort, Union-Find
7. **Dynamic Programming** — Memoization, Tabulation, DP Patterns
8. **Binary Search** — Classic Search, Rotated Arrays, Search Space
9. **Backtracking** — Permutations, Combinations, N-Queens, Pruning
10. **System Design Basics** — Scalability, Load Balancing, Caching, CAP Theorem

**בנק של 75 בעיות** מ-LeetCode עם פתרונות מלאים בעברית, ניתוח סיבוכיות, והסברים.

### 🔷 טראק 3: AI & Machine Learning (8 שיעורים)
**משך כולל:** ~2.5 שעות | **בעיות:** 40 שאלות

למי שחלום שלהם הוא ML Engineer:

2. **Neural Networks** — Perceptron, Backpropagation, Activation Functions, Gradient Descent
3. **NLP & Transformers** — Word Embeddings, Attention, BERT, GPT, Tokenization
4. **Computer Vision** — CNN, Convolution, Pooling, Object Detection, Transfer Learning
5. **LLMs & Prompt Engineering** — Fine-tuning, RAG, Embeddings, Evaluation
6. **MLOps** — Model Deployment, Monitoring, CI/CD, Feature Store, A/B Testing
7. **ML Coding Challenges** — Linear Regression, K-Means, Decision Tree, Gradient Descent
8. **System Design for ML** — Recommendation Systems, Search Ranking, Fraud Detection

**בנק של 40 שאלות** עם קוד Python מלא, דיאגרמות System Design, וסיכומים מעמיקים.

### 🔷 טראק 4: System Design (6 שיעורים)
**משך כולל:** ~2 שעות

עיצוב מערכות בעברית — מה שמבדיל בין Senior Developer ל-Staff Engineer:

1. **יסודות System Design** — Scalability, CAP Theorem, Load Balancing, Caching, Sharding, Message Queues
2. **URL Shortener Design** — כמו bit.ly (Hash Functions, Redis, Analytics)
3. **Chat System Design** — כמו WhatsApp (WebSockets, Message Queues, Real-time)
4. **News Feed Design** — כמו Facebook (Fan-out, Caching, Ranking)
5. **Video Streaming** — כמו YouTube (Transcoding, CDN, Adaptive Bitrate)
6. **Search Engine** — Crawling, Indexing, Ranking, Autocomplete

כל מערכת בנויה עם framework ברור: דרישות, הערכת scale, high-level design, deep dive, ו-trade-offs.
